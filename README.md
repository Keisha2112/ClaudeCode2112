# Сайт компании «Гармония»

Одностраничный сайт строительно-девелоперской компании с вкладками
**Главная**, **О компании**, **Проекты** и **Контакты**. Флагманский
проект — ЖК «Гармония 6».

Весь сайт — это один файл [`index.html`](./index.html): без сборки и
зависимостей, без внешних ресурсов (шрифты, картинки и скрипты не
подгружаются извне). Открывается двойным кликом и разворачивается на
любом статическом хостинге.

## Как посмотреть

Откройте `index.html` в браузере — двойным кликом или перетащив в окно.

## Структура

- **Вкладки** переключаются без перезагрузки (адрес меняется на `#home`,
  `#about`, `#projects`, `#contacts` — ссылки можно давать сразу на вкладку).
- **Проекты** выводятся из объекта `PROJECTS` в конце файла, в `<script>`.
  Клик по карточке открывает окно с характеристиками проекта.
- Дизайн адаптивный: на телефоне меню сворачивается в «бургер».

## Как заменить заглушки данными из презентации

Тексты и характеристики сейчас — **временные заглушки**. Места для замены
помечены в коде словом `TODO`:

1. **Проекты и ЖК «Гармония 6»** — правьте объект `PROJECTS`
   (`index.html`, раздел «ДАННЫЕ ПРОЕКТОВ» внизу файла): название, статус,
   характеристики (`specs`), описание (`desc`), инфраструктуру (`infra`).
2. **О компании, цифры, слоган** — в разметке вкладок `#home` и `#about`.
3. **Контакты** — телефон, e-mail и адрес во вкладке `#contacts`.
4. **Фото и рендеры** — сейчас это CSS-заглушки. Чтобы поставить реальные
   изображения, замените блоки-плейсхолдеры на `<img>` (например,
   `.hero-visual`, `.about-visual`, `.proj .thumb`).
5. **Приём заявок** — форма пока показывает сообщение об успехе локально;
   подключите отправку на почту/CRM (пометка `TODO` в обработчике формы).

## Публикация (GitHub Pages)

Так как это один статический файл, подойдёт любой хостинг. Для GitHub Pages:
включите Pages в настройках репозитория и укажите папку с `index.html`.
